The History of Enhanced Zeus Modules

Enhanced Zeus Modules was originally created back in 2019 by GamesByChris. Since then many people have worked on EZM such as M9-SD, Expung3d, and many more.
EZM has always been meant to be used in Public Zeus to enhance the vanilla Zeus abilities. We've felt that Zeus was lacking many features that have been needed for a long time. From 2019 until 2022 EZM was a private script that only a select few had access to. In early Feb. 2022 Expung3d re-wrote EZM from the ground up for use with Scripted Compositions in Arma 3 Official Zeus servers, creating EZM Lite. With over 100 new modules added, new factions being added, and quality of life changes that Zeus sorely needed, EZM is surely the best in class Vanilla Zeus Enhancement script in the market.

In-Game Keybinds

Zeus Interface Keybinds:
Ctrl + Y: Closes the Zeus interface
Space: Teleports your player to the middle of your screen
Ctrl + U: Opens a menu to change your side
Alt + Left Mouse Button: Controls the selected unit

Normal Interface Keybinds:
Y: Re-opens the Zeus interface
H: Holsters your weapon

Additional Features

EZM adds many new features for Zeuses. Such as a revamped object attributes dialog, shortcuts to commonly used modules, and a controllable unit for Zeus if one doesn't exist.

AI Fixes:
AI placed cannot run out of magazines
AI placed are automatically set to stand
AI placed will have their identity set correctly for the region

Additional Features:
Revamped Object Attribute Menus
Fixes for some Zeus glitches
Quality of life changes for Zeus as a whole
When initially running the script you can choose to create a new controllable Zeus unit
