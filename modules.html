Modules List and Description

AI Modifiers change the AI settings or allow you to otherwise modify AI.

Easy Mode:
- Easy Mode is a toggle. It sets the AI skill to the lowest possible and forces them to stand.

Garrison Modules:
- Garrisoning the AI will make them enter the nearest buildings and get in positions to defend them. Instant teleports them immediately, search makes the AI walk to their positions.

Enable/Disable Lasers:
- These modules force all currently alive AI to turn on or off their lights or lasers on their gun.

Make Hostage:
- This module when placed on a unit will make them into a hostage. Restraining them and adding an action to them that allows players to release them. When the hostage is killed a global message appears showing who killed the hostage.

Make HVT:
- This module when placed on a unit will make them into a HVT. When the HVT is killed all players will be alerted of who killed them. This is useful for elimination missions as it does not show their current location.

Set Ambient Animation:
- This module when placed on a unit will make them play the selected animation until they are stopped with the module or killed.

Set Difficulty:
- This module changes the AI skill of all currently alive AI units. Easy is 0 skill, Medium is 0.5 skill, Hard is 1.0 skill.

Set Stance:
- This module changes the forced stance of all currently alive AI units.

Toggle Unit Carelessness:
- This module when placed on a unit will make them careless. This means they will not engage hostiles and will not react to gunfire.

Toggle Unit Surrender:
- This module when placed on a unit will make them surrender, throwing their gun to the ground.

AI Supports create AI units that are meant to either support the playerbase or act as enemies.

Airdrops:
- Airdrops call an aircraft to do an airdrop on the module's position.

Evac Helicopter:
- Evac Helicopters fly to the modules position, then to a secondary position you select to pickup units, either AI or players. Setting the "Number of Passengers" parameter changes the amount of passengers needed before the helicopter leaves.

Call Reinforcements:
- When placed a helicopter will fly to the modules position and land to drop off troops, then the troops will move to the second position you selected. You can modify the type of reinforcements and the number in the menu created. Unit camos are map dependent.

The Arsenal Creator lets the Zeus create an arsenal for the ground units.

Create Full Arsenal:
- This module changes the object its placed onto, or if given no object, creates an arsenal that allows players to save their loadouts for respawning.

The Automated Missions system creates automated missions to buy the Zeus more time. Currently only configured for Altis.

Helicrash Missions:
- This mission spawns a downed Ghosthawk, surrounded by CSAT in a randomized location. After 15 minutes of the mission being spawned, it will be considered failed if the CSAT is not eliminated. Upon failure or completion another will spawn in 10 minutes unless the system is turned off.

Convoy Missions:
- This mission spawns an enemy CSAT or AAF convoy. To complete the mission, eliminate the convoy troops and secure the truck. Allowing the convoy to reach its destination or destroying the truck is a mission failure. Upon failure or completion another will spawn in 10 minutes unless the system is turned off.

The Clean-Up Tools allow Zeuses to frequently clear the map of garbage and other things that reduce server and client performance.

Delete Clutter:
- This module deletes all dead objects on the map, jet canopys that've been broken off, and items on the floor.

Delete Empty Groups:
- This module deletes all empty groups. Sometimes groups are created that don't automatically delete when empty, this can cause Zeuses to have problems spawning specific sides.

Delete Everything:
- This module deletes all objects on the map.

Delete Markers:
- This module deletes all markers on the map.

Delete Mines:
- This module deletes all mines on the map.

Delete Radius:
- This module deletes all objects within the specified radius from the module placement position.

Remove Protection Zones:
- This module deletes all protection zones on the map.

Delete Ships deletes all the aircraft carriers and destroyers on the map.

Delete All Carriers:
- This module deletes all Aircraft Carriers.

Delete All Destroyers:
- This module deletes all Destroyers.

Developer Tools give advanced tools for those who know how to use them.

Animation Viewer:
- This module opens the Animation Viewer.

Local Debug Console:
- This module opens a debug console that is passed the entity its placed on and the module position.

Find in Config Viewer:
- This module opens the Config Viewer to the Config of the object its placed on.

Function Viewer:
- This module opens the Function Viewer.

GUI Editor:
- This module opens the GUI Editor.

Enviroment modules give Zeuses control over the environment they're building.

Change Time:
- This module changes the current time.

Edit Weather Conditions:
- This module opens a menu that lets you change the weather conditions very specifically.

Explosives create traps or obstacles for the playerbase to move around.

Create Minefield:
- This module creates a minefield with a number of mines in a radius.

Create IED:
- This module creates an IED that detonates when a specific side goes near it. It has many different disguises to keep the players on their toes.

Messages create on screen dialog messages that the players can see.

Send Server Message:
- This module creates large text with the sender name and the message to all players of the specified side.

Send Subtitle Message:
- This module creates default Arma 3 Subtitle text for all players of the specified side.

Object Modifiers let Zeus edit properties of the object.

Attach to Nearest & Detach:
- These modules attach the object to the closest object to it or detach all objects from the object.

Edit Object Attributes:
- This module creates a menu that lets you change the lock state of an object, the textures, simulation, god mode, and gives you an init field to run code in.

Toggle Simulation:
- This module toggles on or off the PhysX simulation for the object.

Toggle Invincibility:
- This module toggles on or off whether damage is enabled for the object.

Hide Object Modules:
- These modules toggles hide or un-hide the object.

Player Modifiers let Zeus edit properties of the player.

Change Side:
- This module changes the unit to a specified side.

Disarm:
- This module removes the weapons from a player.

Heal & Revive:
- This module heals the player and revives them if it can.

Force Eject:
- This module ejects all occupants from the vehicle they're in.

Mute & Un-Mute:
- These modules toggles whether players other than Zeus can speak or not.

Server Settings lets Zeus change server side settings.

Change Side Relations:
- This module changes the relations of BLUFOR, OPFOR, INDEPENDENT, and CIVILIANS.

Disable Mortars:
- This module disallows players from entering Mortars.

Set Respawn Timers:
- This module changes the time between respawns.

Remove Team-Killers:
- This module removes the team-killer status from all players. The team-killer status is applied automatically and disallows players from entering vehicles with the team-killer and makes friendly AI target them.

Special Effects lets the Zeus create special visual effects.

Fire:
- This module creates a fire pillar on the position.

Smoke:
- This module creates a smoke pillar on the position.

Earthquake:
- This module causes an earthquake to happen.

Toggle Lamps:
- This module turns on or off the lamps within the radius.

EMP:
- This module turns off all lamps within the radius and causes car alarms to go off.

Terrain Object Modifiers let Zeus modify the terrain objects that are otherwise uneditable.

Edit Building Doors:
- This module allows for doors to be locked or unlocked on a building.

Hide Terrain Objects:
- This module hides all terrain objects within a radius.

Teleportation allows the Zeus to teleport players to a position.

Teleport Self:
- This module teleports the Zeus to the position.

Teleport All Players:
- This module teleports ALL players to the position.

Teleport One Player:
- This module teleports the selected player to the position.

Teleport Side:
- This module teleports all players of a specific side to the position.

Utilities is mostly used to add objects to the Zeus interface.

Add Objects to Interface:
- This module adds ALL objects to the Zeus interface.

Add Objects to Interface (Radius):
- This module adds ALL objects within the radius to the Zeus interface.

Toggle Auto-Add to Interface:
- This module toggles off the Auto-Add to Interface tool. When turned on everytime the Zeus interface is opened it will add all objects to the interface.

Vehicle Modifiers allow Zeus to change attributes about the vehicle.

Edit Vehicle Attributes:
- This module allows Zeus to change the appearance of the vehicle.

Unflip Vehicle:
- This module unflips the vehicle.

Rearm:
- This module rearms the vehicle.

Refuel:
- This module refuels the vehicle.

Repair:
- This module repairs the vehicle.
